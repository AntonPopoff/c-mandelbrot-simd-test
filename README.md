# SIMD Mandelbrot Set Renderer [WIP]

A **work in progress** Mandelbrot set renderer implemented in C, leveraging SIMD instructions for performance.

## Supported SIMD Instructions

* **SSE4.1+**
* **AVX256**

## Planned Support

* ARM NEON
